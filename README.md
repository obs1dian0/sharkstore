## 📝 Описание проекта
Telegram-бот для интернет-магазина **SharkStore**, написанный на Python с использованием библиотеки `aiogram 3`.

Бот позволяет пользователям просматривать каталог товаров (игры Steam, подписки на нейросети, игровые сервисы), добавлять их в корзину, применять промокоды и оформлять заказы. 

Администратор может управлять ассортиментом прямо через бота: добавлять новые товары (с ключами) и услуги (с инструкциями), редактировать цены, описания и фото, а также создавать промокоды. Все данные надежно хранятся в локальной базе данных SQLite (`db.db`).

---

## ✨ Возможности
* 🖼 **Просмотр каталога** товаров с фотографиями и кастомными описаниями.
* 🛒 **Умная корзина:** добавление товаров и применение скидочных промокодов.
* 💳 **Оформление заказа** с автоматической выдачей цифрового ключа или инструкции к услуге.
* 👤 **Профиль пользователя:** проверка баланса и история заказов.
* 📞 **Связь с поддержкой** администратора напрямую через бота.
* 🛠 **Панель администратора** для полного управления товарами и промокодами прямо в Telegram.

## 🗄 Таблицы базы данных

| Таблица | Назначение |
| :--- | :--- |
| **`users`** | Данные пользователей (баланс, статус, роль) |

## Инструкция по запуску на Windows
1. Клонирование / копирование проекта

Если проект уже есть на рабочем столе, этот шаг можно пропустить.

Если используете GitHub:

git clone https://github.com/ivan-artemev24/python-bot
cd python-bot

2. Создание и активация виртуального окружения

# Создать виртуальное окружение (папка venv появится в корне проекта)
python -m venv venv

# Активировать окружение
.\venv\Scripts\Activate.ps1

При успешной активации в начале строки PowerShell появится префикс (venv).

    Если PowerShell выдаёт ошибку выполнения скриптов, выполните один раз:

    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

3. Установка зависимостей

pip install --upgrade pip
pip install -r requirements.txt

4. Настройка токена

Создайте файл config.py в корне проекта:

TOKEN = "ваш_токен_от_BotFather"

Токен можно получить у @BotFather в Telegram.
5. Запуск бота

python main.py

Бот запустится в режиме long polling. Для остановки нажмите Ctrl+C.
Требования

    Python 3.10+
    Токен бота от @BotFather
    Зависимости из requirements.txt:

aiogram==3.26.0
