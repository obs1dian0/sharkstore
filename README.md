Описание проекта
Telegram-бот для автоматизированной продажи цифровых товаров (игры Steam, подписки на нейросети, игровые сервисы PlayStation/Xbox), написанный на Python с использованием библиотеки aiogram 3.

Бот позволяет пользователям удобно просматривать каталог (с фотографиями и описаниями), искать игры через парсинг Steam, добавлять товары в корзину, применять промокоды и просматривать историю покупок.

Реализована продвинутая панель администратора: администратор может прямо в боте добавлять новые товары (с ключами) и услуги (с инструкциями), редактировать цены, описания и фото существующих товаров, а также генерировать промокоды. Все данные надежно хранятся в локальной базе данных SQLite, которая обновляется автоматически.

Возможности
Умный каталог товаров: Динамическая генерация меню из базы данных с разделением на категории (Игры, Подписки, Аккаунты).

Интеграция со Steam: Парсинг актуальных цен, описаний и скриншотов игр по ID или названию.

Умная корзина и промокоды: Добавление нескольких товаров, применение скидочных промокодов, автоматический перерасчет итоговой суммы.

Разделение логики выдачи:

Товары: автоматическая выдача цифровых ключей из наличия.

Услуги: выдача инструкций покупателю и отправка уведомлений администратору (например, для смены региона).

Многофункциональная Админ-панель (через FSM):

Добавление новых товаров (Steam и ручные категории).

Загрузка фотографий и кастомных описаний.

Редактирование и удаление существующих товаров прямо через список кнопок в чате.

Создание скидочных промокодов.

Локальное хранилище медиа: Отображение локальной картинки (main.jpg) в главном меню бота.

├── main.py              # Основной файл бота (меню, корзина, роутинг)

├── db.db                # База данных SQLite (создается автоматически)

└── requirements.txt     # Зависимости проекта
Таблицы базы данных
Таблица	Назначение
users	Данные пользователей (Telegram ID, баланс, права администратора)
items	Товары/Услуги (название, цена, остаток, категория, фото, описание, тип товара, инструкции)
item_keys	Загруженные цифровые ключи, привязанные к конкретным товарам
orders	Корзина (статус 0) и история оплаченных заказов пользователей (статус 1)
promos	Промокоды (секретное слово и сумма скидки в рублях)
Инструкция по запуску на Windows
1. Клонирование / копирование проекта
Если проект уже есть на рабочем столе, этот шаг можно пропустить.
Если используете GitHub:

PowerShell
git clone https://github.com/ваша-ссылка/sharkstore-bot
cd sharkstore-bot
2. Создание и активация виртуального окружения

PowerShell
# Создать виртуальное окружение (папка venv появится в корне проекта)
python -m venv venv

# Активировать окружение
.\venv\Scripts\Activate.ps1
При успешной активации в начале строки PowerShell появится префикс (venv).

Если PowerShell выдаёт ошибку выполнения скриптов, выполните один раз:

PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
3. Установка зависимостей

PowerShell
pip install --upgrade pip
pip install -r requirements.txt
4. Настройка токена
Создайте файл config.py в корне проекта и укажите ваши данные:

Python
TOKEN = "ваш_токен_от_BotFather"
ADMIN_ID = 1626312647  # Ваш Telegram ID (цифрами, без кавычек)
Токен можно получить у @BotFather в Telegram. Узнать свой ID можно через @getmyid_bot. Также не забудьте поместить картинку main.jpg в папку с проектом.

5. Запуск бота

PowerShell
python main.py
Бот запустится в режиме long polling. Для остановки скрипта нажмите Ctrl+C.
